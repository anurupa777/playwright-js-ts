# playwright-js-ts
This repo contains programming about JS and TS and playwright automation
Prerequisites
Node.js installed (version 14 or higher)
Git for version control
Repository Structure
chapter_01_Basics/: Basic JavaScript setup and verification

01_Basics.js: Basic JavaScript concepts
03_JS_Verify_Setup.js: Setup verification script
chapter_02_JavaScript_Concepts/: JavaScript fundamentals

05_JS_Basics.js: Basic variable declarations
06_Identifier_Rules.js: Identifier naming rules
chapter_03_Identifier_literals/: Advanced JavaScript topics

06_Identifier_Rules.js: Identifier rules (duplicate?)
07_JS_All_Examples.js: Combined examples
08_Identifier_Part2.js: Identifier casing examples
09_Comments.js: Comment examples
10_JS_Shortcuts.js: JavaScript shorthand patterns
VS_Code_Keyboad_Shortcuts_Win.md: Visual Studio Code shortcuts for Windows
How to Run
Clone the repository:

git clone https://github.com/MeghsSm23/LearnPlayWrightBatch2X.git
cd LearnPlayWrightBatch2X
Run any JavaScript file using Node.js:

node chapter_01_Basics/01_Basics.js
